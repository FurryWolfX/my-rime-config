# 自用定制化输入方案

考虑到国内用户访问 github 太慢，仓库迁移至 [gitee](https://gitee.com/wolfx/my-rime-config)

## 支持的输入模式

1. 全拼(小词库版主要用于反查)
2. 燃虎音形(我的主用方案: 自然码双拼+虎码辅助码方案，基于墨奇魔改)
3. 虎码(支持字根展示和全拼反查，基于官方虎码和胖虎魔改)
4. 英文单词模式(再也不怕忘记单词怎么拼了)
5. 郑码(备用)
